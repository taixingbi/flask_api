SwipeToSpin Programming Test (Python)

Assignment

Write a Python program that prompts the user to enter an integer. If the input value is not an integer greater than one, display a helpful error message. Otherwise, output “prime” if the integer is prime (divisible only by 1 and itself), or “composite” if the integer is not prime.

Rules

  • Work on your own. Don’t ask others for help, either in person or electronically.
  • It’s okay to refer to documentation or Google for specific questions about syntax and built-in functions, but not for other people’s solutions to similar programming problems.
  • You will have 24 hours to submit your answer.
  • You will be evaluated on the accuracy, efficiency and elegance of your solution.

Submitting your answer

Submit your finished program by email. If your program consists of more than one file, please combine into a single ZIP file.
